Links

Links are used as navigational elements. They may appear on their own, within a sentence or paragraph.

Button Dropdown

Button Dropdowns present a list of options from which a user can select one option.

Forms and Inputs

Checkbox

Checkboxes are used when there are multiple items to select in a list.

Radio

Radio buttons are group of mutually exclusive choices and only one selection from the group is allowed.

Date-Time picker

Date and time pickers allow users to select a single or a range of dates and times.

Dropdown list

It is list of options for selection. A selected option can represent a value in a form, or can be used as an action

File Uploaders

File uploaders allow users to select one or more files to upload to a specific location.

Input

Text inputs enable the user to interact with and input content and data.

Search

Search enables users to specify a word or a phrase to find relevant pieces of content without the use of navigation.

Floating Labels

A floating label is a text label which appears inside the input field at full font-size

Range Slider

It is an input field that merchants can use to select a numeric value within a given range (min and max values).

Navigational

Breadcrumbs

Breadcrumbs are a list of links representing the current page and its “ancestors”.

Pagination

Pagination allows you to divide large amounts of content into smaller chunks across multiple pages.

Tabs

Tabs are a way of navigating between multiple panels, reducing clutter and fitting more into a smaller space.

Icon

Icon is a graphic symbol that represents some real, fantasy or abstract motive, entity or action.

Content / Data Display

Accordion

It is list of collapsible panels with clickable heading or down arrow, expands to show rolled up content.

Avatar

Avatar is known as a visual form representing a user oneself and one's identity wished.

Card

A card is a small rectangular module with images and text. It is an entry point for users to learn more details.

Charts

A Chart is an embedded component inserted within a worksheet. use a Chart component to better visualize and present your data

Carousel

It is referred as the "slider" or "slides". Carousel works by rotating a group of images that can house text, static images or video content.

Popover

Popovers are used to provide supplemental, useful, unique information about an element, including one or multiple actionable elements.

List Group

Lists are vertical groupings of related content. List items begin with either a number or a bullet.

Table

Tables are a designed for displaying large amounts of data in rows and columns.

Tree

A tree is visualization of a structure hierarchy. A branch can be expanded or collapsed.

Tooltip

A tooltip is a message box that is displayed when a user hovers over, clicks or gives focus to a UI element.

Feedback

Alert

An alert helps to keeps people informed of the status of the system and may or may not require the user to respond.

Progress bar

Progress bars show continuous progress through a process, such as a percentage value.

Toast Message

Toasts are time-based window elements used to display short messages. It will disappear after a few seconds.

Status

Tags

Tags are used to label, organize, or categorize objects. Tags can also serve as a label for content or display counts.

Badge

Use the Badge Component Symbol to draw attention to another interface element or to display a notification.

Structure

Offcanvas

Offcanvas is a sidebar that can be toggled via JavaScript to appear from the left, right, or bottom edge of the viewport.

Scrollspy

Scrollspy is a navigation mechanism that automatically highlights the nav links based on the scroll position

Modal Popups

It is a dialog box/popup window that is displayed on top of the current page
